# D Company Compliance Checklist

Prepared for: D Company ERP / cafe operations in Kerala, India
Review date: 2026-06-15
Use this with a CA, local body licence consultant, and advocate. This is an operational checklist, not legal advice.

## How To Use This

Create a compliance folder in Google Drive or locally:

- `01_business_registration`
- `02_tax_gst_income_tax`
- `03_local_licences`
- `04_food_safety_fssai`
- `05_fire_safety_building`
- `06_labour_payroll`
- `07_environment_waste`
- `08_events_music_gaming_hookah`
- `09_erp_reports_audit`
- `10_renewals_and_returns`

For every item below, record:

- Status: `Done / Pending / Not Applicable / Needs CA Review`
- Licence number
- Issue date
- Expiry date or next review date
- Uploaded proof location
- Responsible person

## Critical Before Opening

- [ ] Confirm legal business structure: proprietorship, partnership, LLP, or private limited.
- [ ] Business PAN available.
- [ ] Business bank account opened in the correct legal name.
- [ ] Rental agreement / ownership proof available for premises.
- [ ] Owner NOC from landlord for cafe, kitchen, gaming, events, signage, and any smoking/hookah area if applicable.
- [ ] Local body trade licence / IFTE & OS / D&O-type licence approved for the exact activities.
- [ ] FSSAI registration/licence active under correct food business category.
- [ ] GST registration active if applicable or voluntarily taken.
- [ ] Shop and Establishment registration active.
- [ ] Profession tax registration/payment process set up.
- [ ] Fire safety review completed; NOC obtained if required for building/seating/use.
- [ ] Commercial LPG / electrical safety / fire extinguisher records available.
- [ ] Waste collection and grease/oil disposal process set up.
- [ ] ERP invoice series, GST rates, FSSAI number, branch GSTIN, and state code verified.
- [ ] CA signs off POS GST treatment before first tax filing.

## Business Registration And Identity

- [ ] Decide business type.
  - Proprietorship is simplest but gives least separation between owner and business.
  - LLP/private limited is better if multiple owners/investors are involved.
- [ ] Register entity if partnership/LLP/company.
- [ ] Obtain PAN for entity.
- [ ] Obtain TAN if deducting TDS on salary, rent, contractor, professional fees, etc.
- [ ] Register on Income Tax e-filing portal.
- [ ] Register Udyam/MSME if eligible.
- [ ] Keep ownership proof:
  - Incorporation certificate / partnership deed / proprietorship declaration.
  - PAN.
  - Aadhaar/KYC of owners/directors/partners.
  - Board/partner resolution if applicable.
  - Bank account proof.

## Premises And Local Body Licences

- [ ] Confirm jurisdiction: Panchayat / Municipality / Corporation.
- [ ] Obtain local trade licence / IFTE & OS / D&O-style licence.
- [ ] Ensure licence covers all activities actually done:
  - Cafe / restaurant.
  - Takeaway.
  - Delivery.
  - Kitchen.
  - Gaming lounge.
  - Event screening / ticketed events.
  - Hookah/smoking area if proposed.
  - Packaged product resale if any.
- [ ] Verify building occupancy certificate / building permit use matches commercial cafe use.
- [ ] Obtain signage / advertisement board permission if required by local body.
- [ ] Keep property tax receipt / building number / ownership or lease docs.
- [ ] Keep landlord NOC.
- [ ] Renew local body licence before expiry.

## Food Safety / FSSAI

- [ ] Apply through FoSCoS/FSSAI portal.
- [ ] Verify correct FBO category:
  - Restaurant / cafe.
  - Food service.
  - Storage if separate storage exists.
  - Caterer if outside catering is done.
  - Relabeller/repacker/manufacturer if packaging own branded food.
- [ ] Apply correct FSSAI level from 2026-04-01 thresholds:
  - Registration: turnover up to INR 1.5 crore.
  - State licence: turnover above INR 1.5 crore and up to INR 50 crore.
  - Central licence: turnover above INR 50 crore.
- [ ] Note: FSSAI 2026 reform introduced perpetual validity, but changes in business category, premises, turnover level, ownership, or licence conditions still need review/modification.
- [ ] Display FSSAI licence/registration number at premises and on invoices/menu where required.
- [ ] Keep FSSAI licence copy in compliance folder.
- [ ] Keep food category/menu list matching actual menu.
- [ ] Keep water test report if required.
- [ ] Keep kitchen layout plan if required.
- [ ] Keep medical fitness records for food handlers if required.
- [ ] Keep pest control service records.
- [ ] Keep cleaning schedule records.
- [ ] Keep food temperature/storage logs for perishables.
- [ ] Keep supplier invoices for traceability.
- [ ] Keep training record for food safety supervisor/handlers; FoSTaC training if applicable.
- [ ] If using cloud kitchen/delivery model, confirm FSSAI category covers it.
- [ ] If making packaged products, check labelling, MRP, batch number, manufacturing/expiry date, allergen declaration, nutrition, veg/non-veg symbol, and Legal Metrology requirements.

## GST Checklist

- [ ] Confirm GST registration status.
  - Services threshold is generally INR 20 lakh aggregate turnover for normal states; verify if voluntary registration is better for business credibility.
- [ ] Verify GSTIN is entered in ERP settings.
- [ ] Verify Kerala state code is `32`.
- [ ] Verify branch address and GSTIN match GST certificate.
- [ ] Use a new invoice series for each financial year.
- [ ] Keep invoice numbers continuous; no duplicate, skipped, or edited invoice numbers without audit note.
- [ ] Restaurant/cafe food service:
  - Usually SAC `996331`.
  - Standalone restaurant/cafe service is generally 5 percent GST without ITC.
  - CGST 2.5 percent + SGST 2.5 percent for Kerala in-state supplies.
  - Do not claim ITC on inputs used for 5 percent restaurant service unless CA confirms a different lawful treatment.
- [ ] Takeaway is usually treated as restaurant service at 5 percent without ITC.
- [ ] In-house delivery is usually restaurant service; confirm with CA.
- [ ] Platform delivery through ECOs such as Zomato/Swiggy:
  - GST on restaurant service through ECO is paid by the ECO under section 9(5).
  - ERP should track this separately from normal taxable POS sales.
  - Reconcile platform payout, commission, TDS/TCS, and invoice records monthly.
- [ ] Event tickets:
  - Do not assume restaurant 5 percent.
  - CA should classify each event: entertainment admission, sports screening, gaming, room/venue service, or bundled food package.
  - Apply GST rate accordingly before selling tickets.
- [ ] Gaming lounge:
  - Console/hourly gaming should be reviewed separately from food.
  - Do not offer betting, gambling, online money gaming, casino-style games, or prize gaming without legal advice; specified actionable claims can trigger much higher GST and legal issues.
- [ ] Hookah:
  - Do not treat tobacco/hookah as normal food service.
  - Tobacco/nicotine products can involve GST/cess, COTPA, police/local restrictions, and possible food safety restrictions.
  - Safest rule: no tobacco/nicotine hookah unless written local legal clearance exists.
- [ ] Customer GSTIN capture:
  - For B2B bills, capture customer GSTIN, legal name, address, and place of supply.
- [ ] HSN/SAC review:
  - Food service: SAC 996331.
  - Events/gaming/hookah/packaged goods must be separately reviewed.
- [ ] Monthly GST reconciliation:
  - POS sales total.
  - Aggregator delivery total.
  - Refunds/cancellations.
  - Cash/Card/UPI settlement.
  - GSTR-1 vs GSTR-3B.
  - ERP tax report vs accounting ledger.
- [ ] GST return calendar:
  - GSTR-1 monthly/quarterly depending profile.
  - GSTR-3B monthly by 20th for monthly filers; quarterly filers follow notified 22nd/24th state schedule.
  - QRMP only if eligible and chosen.
  - Annual return if applicable.
- [ ] E-invoicing:
  - Monitor aggregate turnover threshold; if crossing current e-invoice threshold, B2B invoices may need IRN/e-invoice.
- [ ] GST audit trail:
  - No manual deletion of invoices.
  - Credit notes for corrections.
  - Keep cancelled bills with reason.
  - Keep day-end Z report.

  - Purchase ledger.

  - Expense vouchers.
  - Payroll.
  - Inventory records.

  - Cash sales.
  - Cash expenses.
  - Cash deposit.

- [ ] Monthly supplier reconciliation.
- [ ] Monthly P&L review.

## Labour / Payroll / HR

- [ ] Kerala Shops and Commercial Establishments registration completed.
- [ ] Display required notices under labour law.
- [ ] Maintain employee register.
- [ ] Maintain attendance and shift records.
- [ ] Maintain wage register.
- [ ] Issue appointment letters.
- [ ] Define leave/weekly off/overtime policy.
- [ ] Pay minimum wages applicable to role/category.
- [ ] Profession tax registration completed.
- [ ] Deduct/remit employee profession tax as required.
- [ ] EPFO registration if establishment reaches 20 or more employees.
- [ ] ESIC registration if establishment reaches 10 or more persons in applicable notified area and employees fall under wage coverage.
- [ ] Keep salary slips.
- [ ] Keep proof of salary payment.
- [ ] Keep ID/address proof of employees.
- [ ] Keep emergency contact details.
- [ ] Keep staff food safety/handling training records.
- [ ] Keep incident/accident register.

## Fire, Building, Electrical, LPG And Safety

- [ ] Fire & Rescue NOC obtained if building type, seating capacity, height, occupancy, or local licence requires it.
- [ ] Even if NOC is not required, keep:
  - Fire extinguishers with valid refill date.
  - Kitchen Class K/appropriate extinguisher.
  - Emergency exit plan.
  - Exit signage.
  - First aid kit.
  - Staff fire drill/training record.
  - LPG cylinder storage safety.
  - Electrical inspection/maintenance record.
- [ ] Commercial LPG connection and invoices.
- [ ] Chimney/hood cleaning schedule.
- [ ] No blocked exits.
- [ ] No loose wiring.
- [ ] CCTV if required by police/local licence/building rules.
- [ ] Public liability insurance considered.
- [ ] Asset/fire insurance considered.

## Pollution, Waste And Hygiene

- [ ] Check Kerala State Pollution Control Board consent requirement for restaurant/hotel/kitchen category.
- [ ] Obtain Consent to Establish / Consent to Operate if applicable.
- [ ] Grease trap installed if required.
- [ ] Wastewater disposal compliant.
- [ ] Solid waste segregation:
  - Food waste.
  - Plastic.
  - Glass.
  - Paper.
  - General waste.
- [ ] Local body waste collection agreement/receipts.
- [ ] Used cooking oil disposal record; do not reuse oil beyond safe limits.
- [ ] Pest control records.
- [ ] Cleaning chemical storage record.
- [ ] No dumping waste outside premises.

## Gaming Lounge

- [ ] Local trade licence explicitly covers gaming lounge / amusement / console rental.
- [ ] Confirm local police/local body permission if late-night gaming or crowd events are planned.
- [ ] No gambling, betting, wagering, cash prizes, or real-money games without legal advice.
- [ ] GST classification reviewed by CA.
- [ ] Keep console/device purchase invoices.
- [ ] Keep hourly session logs in ERP.
- [ ] Age restriction policy if needed.
- [ ] House rules displayed.
- [ ] Electrical load/fire safety checked.

## Events / Screenings / Tickets

- [ ] Local licence covers events/screenings.
- [ ] Fire/crowd safety checked for event occupancy.
- [ ] Ticket GST treatment approved by CA before selling.
- [ ] Copyright/public performance rights checked for movies, sports screenings, music, and recorded content.
- [ ] If showing copyrighted matches/movies, obtain proper commercial screening rights.
- [ ] If playing music/DJ/live performance, obtain applicable music licence/permission.
- [ ] Maintain event ticket register:
  - Event name.
  - Date/time.
  - Ticket number.
  - Customer.
  - Tax rate.
  - Payment method.
  - Refund/cancellation.
- [ ] Police/local permission for large crowd/late night if applicable.

## Music / Copyright

- [ ] If playing background music in cafe, check public performance licence requirements.
- [ ] If playing recorded label music, check PPL or other applicable right holder licence.
- [ ] If live music, DJ, karaoke, cover songs, or events, check IPRS and relevant right holder permissions.
- [ ] Keep invoice/licence copy.
- [ ] Do not assume Spotify/YouTube personal subscription gives commercial public performance rights.

## Hookah / Tobacco / Smoking

High risk area. Do not start this casually.

- [ ] Decide if D Company will completely avoid tobacco/nicotine hookah.
- [ ] If any hookah/smoking is planned, get written legal opinion for Kerala/local jurisdiction first.
- [ ] Confirm local body licence explicitly permits it.
- [ ] Confirm police/health department position locally.
- [ ] Follow COTPA:
  - No sale to persons under 18.
  - No sale within 100 yards of an educational institution.
  - No smoking in public places except legally permitted designated smoking area.
  - Required signage.
  - No tobacco advertising/promotion.
- [ ] If seating capacity is below required threshold, do not create smoking area.
- [ ] No service by employees inside smoking area unless law permits.
- [ ] No tobacco/nicotine product without proper statutory warning/labelling.
- [ ] No e-cigarettes/vapes.
- [ ] Fire safety review specifically for charcoal/electric hookah.
- [ ] Separate ventilation if allowed.
- [ ] ERP should tax/track hookah separately from restaurant food.

## Legal Metrology / Packaging

- [ ] If using weighing scales for sale/stock receiving, get scale verified/stamped if required.
- [ ] If selling packaged goods:
  - MRP.
  - Net quantity.
  - Manufacturer/packer details.
  - Batch/lot.
  - Manufacturing/expiry/best before.
  - FSSAI number for food.
  - GST/HSN treatment.
- [ ] Keep supplier invoices for packaged resale items.

## Payment, POS And ERP Compliance

- [ ] GSTIN, FSSAI, address, phone, invoice prefix, and branch state code verified in ERP.
- [ ] Invoice number cannot be edited by staff.
- [ ] Audit log enabled for:
  - Login.
  - Logout/session.
  - Price changes.
  - Discounts.
  - Refunds.
  - Void/cancel bills.
  - Inventory adjustments.
  - Staff/user changes.
  - Tax setting changes.
  - Report exports.
- [ ] Pricing access locked to Nasih/special protected access only.
- [ ] Audit access locked with password.
- [ ] Daily P&L email summary enabled when email setup is complete.
- [ ] Monthly GST/tax health report reviewed by owner and CA.
- [ ] Backup schedule verified.
- [ ] Owner password rotated and stored securely.
- [ ] Separate login accounts:
  - Nasih.
  - Shemeer.
  - Rafi.
  - Sameer.
- [ ] No shared staff password.
- [ ] Cashier permissions separated from owner permissions.

## Renewal / Return Calendar

Daily:

- [ ] POS close.
- [ ] Cash close.
- [ ] Void/refund review.
- [ ] Inventory critical item check.

Weekly:

- [ ] Supplier bills uploaded.
- [ ] Waste/oil/pest/hygiene log reviewed.
- [ ] Low stock report reviewed.
- [ ] Staff attendance verified.

Monthly:

- [ ] GST sales reconciliation.
- [ ] GSTR-1/GSTR-3B preparation.
- [ ] Payroll and profession tax review.
- [ ] EPF/ESI payment if applicable.
- [ ] Bank reconciliation.
- [ ] P&L reviewed.
- [ ] Audit log reviewed.

Quarterly:

- [ ] CA review of GST returns, TDS, tax estimate.
- [ ] Licence expiry calendar review.
- [ ] Fire/safety drill.
- [ ] Supplier GSTIN reconciliation.

Half-yearly:

- [ ] Profession tax half-yearly compliance review.
- [ ] Employee records audit.
- [ ] Pest control/fire/electrical safety file audit.

Yearly:

- [ ] Income tax return.
- [ ] Financial statements.
- [ ] Audit/tax audit if applicable.
- [ ] Local licence renewal.
- [ ] Insurance renewal.
- [ ] FSSAI modification review if turnover/category changed.
- [ ] Shops and Establishment renewal/change review.
- [ ] GST annual return if applicable.
- [ ] New invoice series from 1 April.

## Owner Sign-Off

Before opening day, the owner should sign:

- [ ] I know which activities are legally approved.
- [ ] I know which activities are not approved and must not be done.
- [ ] CA has reviewed GST treatment for food, delivery, events, gaming, and hookah.
- [ ] Local consultant has reviewed licences for the exact premises.

- [ ] Staff have separate logins.
